# Atajos de Teclado - 4ndy-bspwm

> Configuración optimizada para macOS host + Parallels VM

## Aplicaciones

| Atajo | Alternativa | Acción |
|-------|-------------|--------|
| `Super + Enter` | `Alt + Enter` | Abrir terminal (kitty) |
| `Super + D` | `Alt + D` | Abrir lanzador (Rofi) |
| `Super + Shift + F` | - | Abrir Firefox |
| `Super + Shift + B` | - | Abrir Burpsuite |

## Gestión de Ventanas

| Atajo | Alternativa | Acción |
|-------|-------------|--------|
| `Super + W` | `Alt + W` | Cerrar ventana |
| `Super + Shift + W` | `Alt + Shift + W` | Matar ventana (forzar cierre) |
| `Super + M` | - | Alternar modo mosaico/monocle |
| `Super + G` | - | Intercambiar con ventana más grande |
| `Super + Y` | - | Enviar nodo marcado a preselección |

## Estados de Ventana

| Atajo | Alternativa | Acción |
|-------|-------------|--------|
| `Super + T` | `Alt + T` | Modo mosaico (tiled) |
| `Super + Shift + T` | `Alt + Shift + T` | Modo pseudo-mosaico |
| `Super + S` | `Alt + S` | Modo flotante |
| `Super + F` | `Alt + F` | Pantalla completa |

## Flags de Ventana

| Atajo | Acción |
|-------|--------|
| `Super + Ctrl + M` | Marcar ventana |
| `Super + Ctrl + X` | Bloquear ventana |
| `Super + Ctrl + Y` | Ventana pegajosa (sticky) |
| `Super + Ctrl + Z` | Ventana privada |

## Navegación de Ventanas

| Atajo | Alternativa | Acción |
|-------|-------------|--------|
| `Super + ←↓↑→` | `Alt + ←↓↑→` | Enfocar ventana en dirección |
| `Super + Shift + ←↓↑→` | `Alt + Shift + ←↓↑→` | Intercambiar ventana en dirección |
| `Super + P` | - | Enfocar nodo padre |
| `Super + B` | - | Enfocar nodo hermano |
| `Super + ,` | - | Enfocar primer hijo |
| `Super + .` | - | Enfocar segundo hijo |
| `Super + C` | - | Siguiente ventana en escritorio |
| `Super + Shift + C` | - | Anterior ventana en escritorio |
| `Super + [` | - | Escritorio anterior (mismo monitor) |
| `Super + ]` | - | Escritorio siguiente (mismo monitor) |
| `Super + `` ` `` | - | Última ventana enfocada |
| `Super + Tab` | - | Último escritorio |
| `Super + O` | - | Ventana anterior en historial |
| `Super + I` | - | Ventana siguiente en historial |

## Preselección

| Atajo | Acción |
|-------|--------|
| `Super + Ctrl + Alt + ←↓↑→` | Preseleccionar dirección para nueva ventana |
| `Super + Ctrl + 1-9` | Preseleccionar ratio (0.1 - 0.9) |
| `Super + Ctrl + Alt + Space` | Cancelar preselección del nodo |
| `Super + Ctrl + Shift + Space` | Cancelar todas las preselecciones |

## Redimensionar y Mover

| Atajo | Acción |
|-------|--------|
| `Super + Ctrl + ←↓↑→` | Mover ventana flotante (20px) |
| `Alt + Super + ←↓↑→` | Redimensionar ventana |

## Sistema

| Atajo | Acción |
|-------|--------|
| `Super + Escape` | Recargar configuración sxhkd |
| `Super + Alt + R` | Reiniciar bspwm |
| `Super + Alt + Q` | Cerrar sesión bspwm |
| `Ctrl + Alt + L` | Bloquear pantalla (i3lock-fancy) |
| `Ctrl + Alt + P` | Apagar sistema (forzado - para VMs) |
| `Ctrl + Alt + R` | Reiniciar sistema |

## Audio

| Atajo | Acción |
|-------|--------|
| `Ctrl + Shift + ↑` | Subir volumen (+5%) |
| `Ctrl + Shift + ↓` | Bajar volumen (-5%) |
| `Ctrl + Shift + M` | Silenciar/restaurar audio |

## Capturas de Pantalla

| Atajo | Acción |
|-------|--------|
| `Print` | Captura completa (guarda en ~/Screenshots) |
| `Ctrl + Print` | Captura con selección (Flameshot GUI) |

## Terminal (Kitty)

| Atajo | Acción |
|-------|--------|
| `Ctrl + Shift + Enter` | Nueva ventana dividida |
| `Ctrl + Shift + T` | Nueva pestaña |
| `Ctrl + Shift + W` | Cerrar pestaña/ventana |
| `Ctrl + Shift + Z` | Zoom panel actual |
| `Ctrl + ←↓↑→` | Navegar entre paneles |
| `Ctrl + Shift + C` | Copiar |
| `Ctrl + Shift + V` | Pegar |
| `F1` / `F2` | Copiar/Pegar buffer A |
| `F3` / `F4` | Copiar/Pegar buffer B |
